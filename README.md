Описание задачи
Модель предсказывает количество дней, необходимых пользователю для достижения уровня B2 в изучении английского языка. Прогноз строится на основе текущего словарного запаса, темпа обучения и метрик качества ответов пользователя
Технические характеристики
Датасет: Расширен до 15,000 записей с использованием техник аугментации данных для повышения устойчивости модели

Алгоритм: RandomForestRegressor (Ensemble Learning)

Параметры: n_estimators=100, max_depth=15

Метрики качества (финальные):
MAE (Mean Absolute Error): +/- 9.2 дня (среднее отклонение прогноза)
R2 Score: 0.97 (точность аппроксимации данных)
Входные данные (Features)
Модель анализирует 13 параметров, разделенных на три группы:

Vocabulary: Общее кол-во слов (unique_words) и распределение по уровням (A1-B2)
Activity: Темп изучения за 7/30 дней (words_day_30d), количество сессий в неделю, текущий стрик
Quality: Средняя точность ответов (avg_acc_30d) и среднее время на ответ

Ключевые факторы влияния (Feature Importance)
Согласно анализу модели, топ-3 фактора, влияющих на результат:

words_day_30d (46.4%) — интенсивность обучения за последний месяц
unique_words (35.8%) — текущий объем словарного запаса
words_b1 (9.1%) — прогресс по словам среднего уровня

Как запустить
Установите зависимости:
pip install -r requirements.txt

Запустите инференс:
python lingue-ml/inference/predict_b2.py
(Скрипт использует предварительно обученную модель b2_model.pkl)
