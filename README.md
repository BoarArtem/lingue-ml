ML Model Documentation: B2 Time Prediction

Описание задачи
Модель предсказывает количество дней, необходимых пользователю для достижения уровня B2
Прогноз строится на основе текущего словарного запаса, темпа обучения и точности ответов

Технический стек
Алгоритм: RandomForestRegressor
Параметры: n_estimators=100, max_depth=15 
Метрики качества(при тестах):
MAE(Ошибка): ~12-15 дней(в среднем)
R2 Score: ~0.98(в среднем)

Входные данные 
Модель принимает 13 параметров, включая:
Vocabulary: Общее кол-во слов + разбивка по уровням (A1-B2)
Activity: words_day_30d(темп за месяц), sessions_week, streak
Quality: avg_acc_30d(точность), avg_time_sec

Ключевые факторы (Feature Importance)
Анализ показал, что на скорость обучения сильнее всего влияют:
Темп изучения: words_day_30d
Текущий уровень: unique_words
Точность ответов: avg_acc_30d

Как запустить
1.  Установите зависимости: 'pip install pandas scikit-learn joblib`
2.  Запустите предсказание: 'python inference/predict_b2.py'
