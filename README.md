Linguo Machine Learning... один из самых потужних сфер в нашем проекте имеющий очень богатый функционал который удивляет абсолютно каждого в нашем коллективе.

На состояние 24 февраля, наша сфера имеет уже достаточно моделей, которые прошли проф. проверку перед тем как попасть в ветку main..

Мы имеем уже целых 3 готовых модели, которые четко показали себя на тестах и уже готовы отрабатывать свои лошадиные силы на сервисе. Этими моделями стали:
1) b2_predictor -  Модель которая поможет пользователю узнать через сколько он сможет достичь высот с нашим продуктом, а именно уровня B2 в изучаемом им языком.
2) vocabulary_expander - Модель которая очень круто дружит с вашими словарями, он умело анализирует слова которые вы записали в свой словарь и уже оперативно готовит свои мозги чтобы дополнить его для вашего эффективного обучения!
3) llm_word_level - Модель которая интегрирована с сервиса groq сможет умело по промпту сказать уровень языка имея само слово и его перевод/значение
4) llm_sentence_generate - Модель которая интегрирована с сервиса groq сможет умело по промпту сгенерировать предложение для пользователя имея слово, перевод и уровень этого слова

Мы очень довольны продуктивной роботой на ML-side и продолжаем удивлять наших ребят новыми результатами!

Как запустить?
1) https://github.com/BoarArtem/lingue-ml.git
2) docker-compose up --build
3) docker run -p 8000:8000 --env-file .env -v lingue-ml-models:/models lingue-ml-ml-service
